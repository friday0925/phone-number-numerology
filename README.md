# 電話號碼命理分析系統

這是一個結合中華命理學的電話號碼分析工具,可以幫助您找到最適合的電話號碼。

## 功能特色

- 🔍 **電話號碼搜尋**: 從中華電信網站搜尋符合特定格式的電話號碼
- 📊 **八大數字磁場分析**: 分析號碼中的吉凶磁場組合
- 🎯 **八十一靈動數**: 計算號碼的整體能量數值
- ☯️ **五行相容性**: 根據出生日期分析五行相生相剋
- 📈 **綜合評分排名**: 自動計算並排序最適合的號碼

## 安裝需求

```bash
pip install playwright
python -m playwright install chromium
```

## 使用方法

### 方法一: 網頁介面 (推薦) 🌐

最簡單的使用方式!

**步驟 1: 啟動後端服務**
```bash
python app.py
```

**步驟 2: 開啟網頁**
在瀏覽器中開啟 `index.html` 或訪問 `http://localhost:5000`

**步驟 3: 輸入資料**
- 輸入手機號碼 (例: 0978-759-196)
- 輸入出生年月日 (例: 1990/09/25)
- 點擊「開始分析」

分析結果會即時顯示在網頁上! ✨

---

### 方法二: 命令列工具

### 1. 搜尋電話號碼

```bash
python cht_crawler.py
```

這會搜尋所有符合格式 `09??-??9-196` 的電話號碼,並儲存到 `found_numbers.txt`。

### 2. 分析找到的號碼

**基本用法** (使用預設出生日期 1990/09/25):
```bash
python analyze_results.py
```

**指定出生日期**:
```bash
python analyze_results.py --birthdate 1995/03/15
```

**分析指定的電話號碼**:
```bash
python analyze_results.py --phone 0978-759-196
```

**同時指定出生日期和電話號碼**:
```bash
python analyze_results.py --birthdate 1995/03/15 --phone 0978-759-196
```

**查看所有參數說明**:
```bash
python analyze_results.py --help
```

分析結果會:
- 在螢幕上顯示排名和前3名的詳細分析
- **自動儲存到桌面**,檔案名稱為電話號碼
  - 單一號碼分析: `0978-759-196.txt`
  - 多個號碼分析: 
    - 總覽報告: `電話號碼分析總覽_N個號碼.txt`
    - 個別報告: `0978-759-196_分析報告.txt` (每個號碼一個檔案)

### 3. 測試分析

```bash
python test_analysis.py
```

### 4. 自訂分析

**方法一: 使用命令列參數**

```bash
# 分析特定號碼
python analyze_results.py --phone 0912-345-678 --birthdate 1990/09/25
```

**方法二: 在 Python 中使用**


## 分析方法說明

### 八大數字磁場

分析號碼中連續兩位數字的組合,包含:

**吉星磁場:**
- 天醫 (13, 31, 68, 86, 49, 94, 27, 72): 財富、智慧、婚姻
- 生氣 (14, 41, 67, 76, 93, 39, 82, 28): 貴人、人際關係
- 延年 (19, 91, 87, 78, 34, 43, 26, 62): 事業、領導力

**凶星磁場:**
- 絕命 (12, 21, 69, 96, 48, 84, 37, 73): 投資、壓力
- 禍害 (17, 71, 89, 98, 46, 64, 32, 23): 口舌是非
- 五鬼 (18, 81, 97, 79, 36, 63, 24, 42): 意外變動
- 六煞 (16, 61, 74, 47, 38, 83, 29, 92): 感情波折

### 八十一靈動數

根據號碼末四位計算整體能量:
- 公式: `(末四位數字 % 80) or 80`
- 對應到 1-81 的吉凶意義

### 五行相容性

根據出生年份計算本命五行,分析號碼中數字的五行分布:
- 數字 1,2 = 木
- 數字 3,4 = 火
- 數字 5,6 = 土
- 數字 7,8 = 金
- 數字 9,0 = 水

## 評分系統

綜合評分 = 磁場分析(40%) + 靈動數(30%) + 五行相容性(30%)

推薦等級:
- ★★★★★ (80-100分): 極力推薦
- ★★★★☆ (70-79分): 非常適合
- ★★★☆☆ (60-69分): 適合
- ★★☆☆☆ (50-59分): 普通
- ★☆☆☆☆ (<50分): 不推薦

## 檔案說明

**網頁介面**:
- `index.html`: 網頁 UI 介面,提供視覺化的分析體驗
- `app.py`: Flask API 後端服務,處理分析請求

**命令列工具**:
- `cht_crawler.py`: 中華電信網站爬蟲,搜尋符合條件的電話號碼
- `phone_numerology.py`: 核心分析模組,包含所有命理計算邏輯
- `analyze_results.py`: 分析腳本,讀取找到的號碼並生成報告 (自動儲存到桌面)
- `test_analysis.py`: 測試腳本,快速測試分析功能

**資料檔案**:
- `found_numbers.txt`: 搜尋結果 (由 crawler 生成)

**輸出檔案** (自動儲存到桌面):
- 單一號碼: `電話號碼.txt` (例: `0978-759-196.txt`)
- 多個號碼: 
  - `電話號碼分析總覽_N個號碼.txt` - 所有號碼的排名和詳細分析
  - `電話號碼_分析報告.txt` - 每個號碼的獨立報告
