# 網頁 UI 使用指南

## 🌐 快速開始

### 1. 安裝依賴

```bash
pip install flask flask-cors
```

### 2. 啟動後端服務

```bash
python app.py
```

您會看到:
```
============================================================
電話號碼命理分析 API 服務
============================================================
伺服器啟動於: http://localhost:5000
API 端點: http://localhost:5000/analyze
請在瀏覽器中開啟 index.html 使用網頁介面
============================================================
```

### 3. 開啟網頁介面

**方法一**: 直接開啟 `index.html` 檔案
- 在檔案總管中雙擊 `index.html`
- 或右鍵 → 開啟方式 → 選擇瀏覽器

**方法二**: 訪問本地伺服器 (如果使用 Flask 提供靜態檔案)
- 在瀏覽器中訪問 `http://localhost:5000`

### 4. 使用介面

1. **輸入手機號碼**: 
   - 格式: `0978-759-196` 或 `0978759196`
   - 系統會自動格式化

2. **輸入出生年月日**:
   - 格式: `1990/09/25`
   - 系統會自動格式化

3. **點擊「開始分析」**
   - 系統會即時顯示分析結果
   - 包含完整的命理分析報告

## 🔧 技術架構

### 前端 (index.html)
- **HTML5**: 語意化標籤
- **CSS3**: 漸層、動畫、響應式設計
- **JavaScript**: 
  - Fetch API 呼叫後端
  - 輸入驗證和格式化
  - 動態 UI 更新

### 後端 (app.py)
- **Flask**: 輕量級 Web 框架
- **Flask-CORS**: 跨域資源共享
- **phone_numerology**: 核心分析模組

## 🐛 常見問題

### Q: 無法連接到伺服器
**A**: 確認後端服務是否已啟動
```bash
python app.py
```

### Q: 出現 CORS 錯誤
**A**: 確認已安裝 flask-cors
```bash
pip install flask-cors
```

### Q: 分析結果不顯示
**A**: 
1. 檢查瀏覽器控制台 (F12) 是否有錯誤訊息
2. 確認輸入格式正確
3. 確認後端服務正常運行

### Q: 想要修改介面樣式
**A**: 直接編輯 `index.html` 中的 `<style>` 區塊

## 📝 自訂設定

### 修改 API 端點

在 `index.html` 中找到:
```javascript
const API_URL = 'http://localhost:5000/analyze';
```

修改為您的 API 位址。

### 修改伺服器埠號

在 `app.py` 中找到:
```python
app.run(debug=True, host='0.0.0.0', port=5000)
```

修改 `port` 參數。
